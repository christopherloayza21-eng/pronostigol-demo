## Título corto

PronostiGol - Sistema web de pronósticos deportivos con historial, estadísticas y simulación de apuestas

## Descripción breve

Desarrollé una aplicación web local para analizar partidos de fútbol usando datos históricos, forma reciente, métricas ofensivas/defensivas, calendario, resultados y simulación de apuestas ficticias. El sistema genera pronósticos 1X2, marcador probable, goles esperados, mercados estadísticos y permite evaluar el rendimiento del modelo contra resultados reales.

## Descripción profesional

PronostiGol es un sistema de análisis deportivo construido con HTML, CSS, JavaScript, Python y SQL Server opcional. La aplicación permite cargar datos de selecciones y clubes, analizar partidos, generar probabilidades, guardar pronósticos previos al encuentro y comparar posteriormente contra resultados reales.

El proyecto incluye:

- Motor de pronóstico con forma reciente, goles esperados, Elo, recencia, localía y contexto del partido.
- Calendario y cuadro de torneo para Mundial.
- Historial de pronósticos con evaluación por mercado.
- Laboratorio de apuestas ficticias con ROI, CLV, cuota justa, edge y control de banca.
- Importación/sincronización de datos desde archivos locales y APIs deportivas.
- Respaldo opcional en SQL Server.
- Separación de archivos sensibles mediante `.gitignore`.

## Stack técnico

- Frontend: HTML, CSS, JavaScript vanilla.
- Backend local: Python.
- Base de datos opcional: SQL Server Express.
- Scripts: PowerShell y Python.
- Control de versiones: Git y GitHub.

## Qué demuestra este proyecto

- Diseño de una aplicación completa de escritorio/web local.
- Manejo de datos deportivos en JSON, CSV y APIs.
- Modelado estadístico básico aplicado a fútbol.
- Persistencia local y respaldo en base de datos.
- Evaluación de predicciones contra resultados reales.
- Organización de proyecto con GitHub.
