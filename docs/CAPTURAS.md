# Capturas recomendadas para Workana

Cuando armes el documento o me pases imágenes, intenta cubrir estas pantallas:

1. Pantalla principal de configuración del partido.
2. Resultado del modelo con probabilidades 1X2.
3. Pronósticos oficiales del partido.
4. Mercados estadísticos: goles, córners, tarjetas, tiros y faltas.
5. Calendario del Mundial o de ligas.
6. Cuadro eliminatorio.
7. Historial de pronósticos con resultados reales.
8. Laboratorio de apuestas ficticias.
9. Ranking o control de rendimiento.
10. Pantalla de ligas/temporadas si quieres mostrar que no es solo Mundial.

## Formato ideal

- Imágenes PNG o JPG.
- Resolución clara, sin datos personales.
- Evita mostrar API keys, rutas privadas o saldos reales.
- Si aparece historial de apuestas, que sea ficticio o demo.

## Cómo las usaríamos

Con tus capturas puedo preparar:

- README más visual.
- Sección de galería en `docs/`.
- Texto de caso de estudio.
- Versión pública/demo más profesional.
- Publicación para Workana o LinkedIn.
