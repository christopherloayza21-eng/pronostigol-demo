# Clave de football-data.org

Para sincronizar ligas automaticamente:

1. Copia `football-data.key.example`.
2. Renombra la copia a `football-data.key`.
3. Pega tu API key adentro.
4. Ejecuta `SINCRONIZAR_LIGAS.bat`.

Ese archivo no debe subirse ni compartirse.

## Competiciones configuradas

El sincronizador usa solo competiciones que aparecen disponibles en el plan gratis de football-data.org:

- Premier League
- Championship
- LaLiga
- Bundesliga
- Serie A
- Ligue 1
- Eredivisie
- Primeira Liga
- Brasileirao Serie A
- UEFA Champions League
- FIFA World Cup
- European Championship

Europa League y Conference League no aparecen en esa lista gratuita. Si luego consigues otra API o un plan que las incluya, se pueden agregar como nuevos codigos sin tocar el motor principal.
