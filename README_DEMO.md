# PronostiGol Demo

Esta carpeta es una version limpia para portafolio. No contiene claves reales,
historial privado, simulaciones personales, cache, ZIPs ni archivos temporales.

## Ejecutar

1. Abre esta carpeta.
2. Ejecuta INICIAR_PRONOSTIGOL.bat.
3. Entra a http://localhost:8000.

## Datos incluidos

- Calendario/base del Mundial.
- Historial internacional compacto.
- Perfiles estadisticos agregados.
- Datos demo de ligas sincronizadas.

## Datos excluidos

- config/football-data.key
- data/*.local.json
- Capturas privadas sin seleccionar.
- Base SQL Server local.
- Archivos temporales.
